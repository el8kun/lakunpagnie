# La Kunpagnie

Site V1 prêt pour GitHub Pages.

## Fichiers

- `index.html` : structure du site
- `style.css` : design / DA
- `script.js` : menu mobile
- `assets/` : dossier pour ajouter tes images, logos, miniatures, BD, cartes, etc.

## Mise en ligne sur GitHub Pages

1. Crée un dépôt GitHub nommé par exemple `la-kunpagnie`.
2. Envoie les fichiers de ce dossier dans le dépôt.
3. Va dans `Settings` > `Pages`.
4. Dans `Build and deployment`, choisis :
   - Source : `Deploy from a branch`
   - Branch : `main`
   - Folder : `/root`
5. Attends quelques minutes.
6. Ton site sera accessible à une adresse du type :

`https://el8kun.github.io/la-kunpagnie/`

## À personnaliser

- Remplacer les liens réseaux.
- Ajouter ton vrai logo dans `assets`.
- Ajouter tes images Capelinho, BD, trophées et cartes.
- Modifier les horaires dans la section Planning.
